📊 HR Analytics Dashboard – Workforce Insights (Power BI)
📌 Project Overview

The HR Analytics Dashboard is an interactive workforce analysis solution built using Microsoft Power BI.

This project provides a comprehensive overview of employee demographics, attrition trends, performance metrics, salary insights, and workforce behavior to support data-driven HR decision-making.

The dashboard is divided into multiple analytical views:

HR Executive Overview

Workforce Demographics

Talent & Performance

Employee Insights

Workforce Movement Summary

🎯 Business Objectives

Monitor overall workforce health

Analyze attrition rate and turnover patterns

Evaluate employee performance distribution

Understand salary trends across job roles

Identify demographic-based workforce insights

Improve HR strategy using data-driven decisions

📊 Key Metrics

1K Total Employees

16.12% Attrition Rate

237 Attrition Count

6.50K Average Monthly Income

3.15 Average Performance Rating
